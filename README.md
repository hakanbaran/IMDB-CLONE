# IMDB Clone;
- It is an application where you can get information about movies and TV Series and watch their trailers. Powered by TMDB API. Build with UIKit...

### Technologies used in the development of the application:
- UIKit
- MVVM
- TabBar
- TableView
- CollectionView
- CoreData
- SDWebImage
- ImageSliderShow
- SerachBar
- TMDB API
- Youtube API
